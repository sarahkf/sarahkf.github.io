About Me

I'm a Computer Science major at Swarthmore college with minors in Philosophy and Statistics. I am especially interested in programming languages and Computer Science education.

Projects

Implemented tuples for Pyret programming language.

Pyret is a programming language designed for programming education used as part of the Bootstrap curriculum and in undergraduate introductory computer science courses. Over summer 2016, I modified Pyret's compiler to enable the use of tuples in the language. I also attended CSPD week as a facilitator for Bootstrap. Visit here to see some examples of tuples in Pyret.

Implemented a compiler in OCaml

As part of a course assignment, I implemented a compiler that had several features including let bindings, function applications, if statements, unary and binary operators, and register allocation. I also built a garbage collector in C. To see this code, please contact me for a link.

Built a Unix Shell

As part of a course assignment, I implemented a Unix Shell. This shell has the ability to run processes in the foreground and the background and also has several built in commands including 'history' and 'exit'. To see this code, please contact me for a link.
