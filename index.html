Selected Projects

I have had the opportunity to work on many different projects. Here are descriptions of just a few.

Implemented tuples for Pyret programming language

Pyret is a programming language designed for programming education used as part of the Bootstrap curriculum and in undergraduate introductory computer science courses. Over summer 2016, I modified Pyret's compiler to enable the use of tuples in the language. I also attended CSPdWeek as a facilitator for Bootstrap. Visit here to see some examples of tuples in Pyret or here to see a more detailed description of my project.

What's Cookin' Good Lookin'

For this project, I used artificial intelligence techniques to attempt to solve the "What's Cooking?" Kaggle Competition. The goal of this competition was to build a kernel that could take in a recipe and return the food category of that recipe. I used algorithms such as Random Forest, Naive Bayes, and K-Nearest Neighbors. To see the code for this project, please contact me for a link.

Compiler implemented in OCaml

As part of a course assignment, I implemented a compiler that has several features including let bindings, function applications, if statements, unary and binary operators, and register allocation. I also built a garbage collector in C. To see the code for this project, please contact me for a link.

Plagiarism Detector

As part of a course assignment, I built a plagiarism detector that compares several documents for similarities. It utilizes either LinkedBSTs or AVLTrees to compare the documents. To see the code for this project, please contact me for a link.
